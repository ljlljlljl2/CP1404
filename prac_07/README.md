## About This Repository
This repository contains my practical work for CP1404.  
It includes weekly programming exercises covering Python fundamentals, clean code, object-oriented programming, file handling, and project development practices.

## About Me
I am a CP1404 student learning how to write cleaner, more maintainable Python code through practical exercises, Git version control, and code reviews.

## Clean Code Lessons I've Learned
- Use meaningful names for variables, functions, classes, and files.
- Break large problems into smaller functions that each have one clear responsibility.
- Avoid repeating code by reusing logic and following DRY principles.
- Keep formatting consistent and follow Python conventions such as PEP 8.
- Test code incrementally and refactor only after the program works.
